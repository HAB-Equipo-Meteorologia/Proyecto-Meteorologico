### Pre-requisitos:
- [Conda](https://docs.conda.io/projects/conda/en/stable/user-guide/install/index.html)

### Setup:
```sh
cd project
conda env create -f environment.yml
conda activate proyecto_meteorologico
```

### Ejecución (UNIX):
```sh
./run.sh
```

### Ejecución (Windows):
```sh
run.bat
```
